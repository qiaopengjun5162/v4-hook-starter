# Uniswap V4 hook

## 实操

```shell
forge init v4-hook-demo
cd v4-hook-demo/
c # code .

forge install Uniswap/v4-core --no-commit
forge install Uniswap/v4-periphery --no-commit
forge remappings > remappings.txt    # 生成remappings.txt
```
